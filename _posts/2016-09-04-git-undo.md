[来源](http://megakemp.com/2016/08/25/git-undo/)


运行

```shell
git config --global alias.undo '!f() { \
    git reset --hard $(git rev-parse --abbrev-ref HEAD)@{${1-1}}; \
}; f'
```

即可

注意这里的 Undo 仅仅是取消 commit 等等，不是真正的 undo。

其实实际上就是 `git reset --hard`
